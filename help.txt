EXPRESS
body-parser: Middleware to capture form data.
EJS module: To create template

If you want to send array of objects or collection of data to EJS Template.
OR want to create to-do app where on tapping add button, input form data get appear on home page or index.ejs file,
below is approach.


PROCESS STEP BY STEP:

1] declare empty array named items.

2] Capture form data.

3] pushing form data to array named items, for every add button click in index.ejs file.
4] Redirect you to home page.

5] Renders homepage on get request for index.ejs.
6] Use for-loop using scriplet tag to  iterate array and print item from array.
